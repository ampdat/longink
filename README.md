Longink: Offline HTML5 feed and web page reader
====================================

Longink reads a feed and stores its associated articles in HTML5 storage for offline reading.

# Technologies
* Coffeescript
* Backbonejs
* JQuery Mobile
* PhoneGap (for the native versions)
* NodeJS
